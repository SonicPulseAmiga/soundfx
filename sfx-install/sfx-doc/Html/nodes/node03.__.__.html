3 Die ARexx Schnittstelle

Der ARexx-Port von SoundFX heißt "REXX_SFX". Über diesen können sie den Funktionsumfang von SoundFX erheblich erweitern. So ist es z.B. möglich SoundFX fernzusteuern oder Sample mit anderen Programmen auszutauschen. Sie können sogar eigenen Effekte in ARexx programmieren.
Wichtig: seit Version 3.70 besitzen alle Befehle, den Prefix "SFX_" um Namenskollisieonen zu vermeiden.
