3.2.3 Fensterfunktion

Folgende Parameter einer Fensterfunktion lassen sich per ARexx ändern. Der benötigte <Präfix> (z.B. W1) ist in den Beschreibungen der Operatoren enthalten.
Parameter Beschreibung
<Präfix>WinType "Rectangle","Bartlett","Fejer","Welch","Hanning","Hamming","Blackman","Kaiser","HalfSine","HalfSineQ","HalfSineS"
<Präfix>WinPar Parameter der Fensterfunktion
