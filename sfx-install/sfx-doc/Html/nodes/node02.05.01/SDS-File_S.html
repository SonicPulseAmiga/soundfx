SDS-File_S
Speichert Sample Dump Standard Dateien.
Loops ja (vorwärts)
Kanäle nein (mono)
Kompressionja (PCM-8,PCM-16,PCM-24,PCM-32)
Dieses Format ermöglicht es Ihnen Samples mit Ihrem Sampler (Profisampler, keine Parallelportsampler) auszutauschen. Dazu benötigen sie weiterhin einen SysEx Dumper. Speichern sie die Datei und senden sie diese an den Sampler per MIDI/SCSI.

Parameter
ChannelMidikanalnummer auf der das Sample gesendet werden soll (0-16).
SampleSpeicherplatznummer auf der der Sampler die Daten ablegen soll.
